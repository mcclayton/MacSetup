# iTerm2

This folder contains the iTerm2 preferences file.

## Installed By

`configureITerm` in `lib/macsetup/appConfigUtil.sh`.

## Files

- `com.googlecode.iterm2.plist` - iTerm2 preferences.

## Destination

MacSetup copies the preferences to:

```sh
~/Library/Preferences/com.googlecode.iterm2.plist
```

## Manual Loading

To load manually from iTerm2:

1. Open `iTerm2 -> Preferences -> General`.
2. Select `Load preferences from a custom folder or URL`.
